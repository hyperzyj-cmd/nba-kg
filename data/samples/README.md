# Data Samples

This folder contains small samples of the pipeline outputs for quick inspection
without downloading the full dataset.

## Files

| File | Description | Full version |
|------|-------------|--------------|
| `crawler_sample.jsonl` | 5 crawled Wikipedia pages | `data/raw/crawler_cleaned.jsonl` |
| `ner_sample.csv` | 50 NER entity records | `data/processed/extracted_knowledge.csv` |
| `kg_sample.ttl` | 100 triples from initial KG | `kg_artifacts/initial_kg.ttl` |

## Full Data

The full datasets are excluded from the repository due to size.
Run the pipeline from scratch using the instructions in the root README.md.
